# 小程序入门
`小程序的布局`
#### 小程序布局常用到的一些标签
###### view:标签相当于div可以设置
```
  <view>我是内容</view>
  可定义class类
```
###### image:标签相当于img可以设置
```
    image双标签
    <image src="src"></image>
```
###### navigator:标签相当于a标签可以设置
```
<navigator url="../index/index">我是跳转
</navigator>
```
